    • REALIZAR EL SIGUIENTE PROGRAMA EN EL LENGUAJE JAVASCRIPT

    Realiza un programa que defina la serie de fibonacci. 0,1,1,2,3,5,8..etc


    CODIGO EN JAVASCRIPT
    < html>
    < head>
    < title> Serie de Fibonacci < / title>
    < / head>
    < body>
    < script type="text/javascript">
    var a=0; var b=1; var c; var i; var x;
    x = prompt("Cuantos numeros quieres que te muestre?");
    document.write(a+", ");
    document.write(b+", ");
    for(i=3;i<=x;i++)
    {
    c=a+b; a=b; b=c; document.write(c+"," );
    }
    < / script>



    Se utilizo el editor de textos notepad++ con HTML con la etiqueta de Script

    ¿por que se te hizo facil?
